PANDUAN NAMA FOTO JPG CHICC STYLE
================================

BACA SATU KALI SEBELUM MULAI
1. Di File Explorer, aktifkan: View > Show > File name extensions.
2. Jangan mengetik atau menempel .jpg saat mengganti nama. Akhiran JPG sudah ada.
3. Salin hanya nama pada kolom SALIN NAMA INI. Jangan salin .jpg.
4. Contoh Stevy / Dusty Pink: ganti nama menjadi stevypink
   Windows akan menyimpannya sebagai stevypink.jpg.
5. Jangan mengetik stevypink.jpg sendiri. Itu dapat menjadi stevypink.jpg.jpg.
6. Masukkan semua foto ke folder images pada website.
7. Setelah selesai, klik dua kali Run Chicc Image Fixer.bat.
   Alat ini memperbaiki .jpg.jpg, mengubah .jpeg menjadi .jpg, merapikan huruf,
   dan mencocokkan banyak nama berdasarkan SKU, produk, dan warna.
   Alat tidak akan menimpa file lain yang sudah ada.
