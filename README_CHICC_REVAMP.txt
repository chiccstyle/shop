PANDUAN PASANG WEBSITE CHICC STYLE
==================================

ISI PAKET
- index.html
  Halaman website utama yang sudah diperbaiki.
- admin-tools\CHICC_IMAGE_RENAME_GUIDE.txt
  Daftar nama foto yang harus dipakai admin.
- admin-tools\chicc-image-map.csv
  Daftar nama foto dalam bentuk tabel.
- admin-tools\chicc-image-fixer.ps1
  Alat otomatis untuk merapikan nama foto JPG.
- admin-tools\Run Chicc Image Fixer.bat
  File yang diklik dua kali untuk menjalankan alat otomatis.
- CHICC_PRODUCT_AUDIT.txt
  Ringkasan hasil pemeriksaan produk dan SKU.

CARA PASANG WEBSITE
1. Simpan salinan website lama sebagai cadangan.
2. Ganti index.html lama dengan index.html dari paket ini.
3. Jangan hapus folder images, favicon.png, apple-touch-icon.png, CNAME,
   robots.txt, sitemap.xml, atau gambar banner yang sudah ada.
4. Upload foto produk ke folder images sesuai nama pada panduan admin.
5. Buka website dari HP dan komputer untuk memastikan tampilannya benar.

CARA ADMIN MENYIAPKAN FOTO
1. Buka folder yang berisi semua foto JPG.
2. Salin ke folder itu:
   - CHICC_IMAGE_RENAME_GUIDE.txt
   - chicc-image-fixer.ps1
   - Run Chicc Image Fixer.bat
3. Klik dua kali Run Chicc Image Fixer.bat.
4. Periksa daftar nama lama dan nama baru yang muncul.
5. Jika sudah benar, ketik YES lalu tekan Enter.
6. Buka chicc-image-rename-log.txt untuk melihat hasilnya.
7. Upload foto yang sudah rapi ke folder images website.

ATURAN PENTING UNTUK ADMIN
- Aktifkan: File Explorer > View > Show > File name extensions.
- Saat mengganti nama manual, jangan mengetik .jpg.
- Contoh yang benar: stevypink
  Windows akan menyimpannya sebagai stevypink.jpg.
- Jika terlanjur menjadi stevypink.jpg.jpg, alat otomatis akan memperbaikinya.
- Alat otomatis tidak akan menimpa file lain yang sudah ada.

CATATAN WEBSITE
- Katalog berisi 65 produk yang disetujui.
- Celana, Rok, Calista, VELA, dan Diana tidak dimasukkan.
- Virly memakai SKU induk A38.
- Ayana memakai SKU induk jsa01.
- Tujuh produk yang mempunyai ukuran tetap memakai pilihan ukuran dan SKU variasi.
- Garis penggaris di bawah header tetap dipakai. Bar layanan hitam sudah dihapus.
- Hadiah Spesial Chicc tidak memakai kode voucher dan tidak perlu disalin.
- Setelah permainan amplop berhasil, hadiah FREE KACA BULAT aktif otomatis saat
  subtotal belanja mencapai Rp100.000 atau lebih.
- Jika foto warna belum diunggah, halaman produk akan menampilkan contoh warna
  dan tulisan pengganti. Foto akan muncul otomatis setelah admin mengunggah JPG
  dengan nama yang benar ke folder images.
- Link Affiliate Program sementara membuka email. Ganti href pada affiliateLink setelah
  alamat website affiliate final sudah tersedia.
